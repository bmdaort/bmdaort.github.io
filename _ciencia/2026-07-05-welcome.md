This section holds notes, paper summaries, and small experiments.

## How to write here

Each entry is a Markdown file in the `_ciencia/` folder. The filename becomes
the URL, so use simple slugs: `my-post.md` → `/ciencia/my-post/`.

### Figures

Put images in `assets/ciencia/<slug>/` and link them with normal Markdown:

```markdown
![Experiment diagram](/assets/ciencia/welcome/diagram.png)
```

### Code and formulas

Triple-backtick code blocks get syntax highlighting automatically:

```python
import numpy as np
np.random.seed(42)
print(np.random.rand(3))
```

### Embedded HTML

You can also paste raw HTML — useful for interactive charts or third-party
embeds:

<p style="text-align:center;color:var(--text-muted);font-size:14px;">
  (an iframe or widget could go here)
</p>
